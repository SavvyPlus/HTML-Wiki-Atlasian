Purpose

The Cost Calculator is a model for calculating utility invoices based on inputs of usage data, rates and applicable market parameters.

Rather than an application in its own right, the Cost Calculator is a calculation engine can be used in a number of ways to provide a range of valuable energy accounting services.

Uses

Uses of the cost calculator have included:

Input DataApplication
Actual usage and ratesRetailer invoice validation
Actual usage and ratesAccruals, cost reporting
Projected usage, actual and/or projected ratesBudgeting, cost outlooks
Actual usage, hypothetical ratesTracking of savings from rate management
...


Technical detail

The Cost Calculator is spreadsheet-based. The key calculations are formula-based, although simple VBA macros are used to loop through invoices and to save results to a database.
